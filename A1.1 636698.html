2- Una variable que hubiera decidido medir si formara parte del grupo de investigación sería el nivel de actividad física, que puede ser una variable cuantitativa o cualitativa, dependiendo de cómo se mida. Se podría medir cuantitativamente registrando el número de horas de actividad física realizadas por semana, o cualitativamente, clasificando el nivel de actividad en categorías como "baja", "moderada" o "alta". Otra variable que hubiera decidido medir sería el consumo de alimentos procesados, que es una variable cualitativa. Una dieta alta en alimentos procesados está relacionada con el aumento de peso y la obesidad. El consumo de estos alimentos podría clasificarse en categorías como "bajo", "moderado" o "alto".

3- En un estudio de inferencia, se utilizaría NivelDeObesidad como la variable dependiente para analizar cómo las demás variables influyen en los diferentes niveles de obesidad. Esta variable es cualitativa, por lo que, en lugar de calcular un promedio como se haría con una variable cuantitativa, se determinaría el número de registros asociados a cada categoría dentro de esa columna mediante la función value_counts().

4- Se utiliza la librería matplotlib para crear un scatter plot. Primero, se seleccionanaron los datos de la columna Edad y NivelDeObesidad del DataFrame data, filtrando solo las filas donde el sexo es femenino. Luego, se genera el gráfico de dispersión utilizando los datos de edad en el eje X y los niveles de obesidad en el eje Y. El parámetro alpha=0.02 ajusta la transparencia de los puntos. Finalmente, se muestra el gráfico con plt.show().

In [61]:
import matplotlib.pyplot as plt  # importa la librería para crear gráficos.
data_x = data.Edad[data.Sexo == "Female"]  # filtra los datos de edad para mujeres.
data_y = data.NivelDeObesidad[data.Sexo == "Female"]  # filtra los datos de nivel de obesidad para mujeres.
plt.figure(figsize=(8,4))  # crea una figura de 8x4.
plt.scatter(x=data_x, y=data_y, alpha=0.02)  # crea un gráfico con los datos filtrados.
plt.show()  # muestra el gráfico.
