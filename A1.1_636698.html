4- Se utiliza la librera matplotlib para crear un scatter plot. Primero, se seleccionanaron los datos de la columna Edad y NivelDeObesidad del DataFrame data, filtrando solo las filas donde el sexo es femenino. Luego, se genera el grfico de dispersin utilizando los datos de edad en el eje X y los niveles de obesidad en el eje Y. El parmetro alpha=0.02 ajusta la transparencia de los puntos. Finalmente, se muestra el grfico con plt.show().

In[61]:
import matplotlib.pyplot as plt  # importa la librera para crear grficos.
data_x = data.Edad[data.Sexo == "Female"]  # filtra los datos de edad para mujeres.
data_y = data.NivelDeObesidad[data.Sexo == "Female"]  # filtra los datos de nivel de obesidad para mujeres.
plt.figure(figsize=(8,4))  # crea una figura de 8x4.
plt.scatter(x=data_x, y=data_y, alpha=0.02)  # crea un grfico con los datos filtrados.
plt.show()  # muestra el grfico.
